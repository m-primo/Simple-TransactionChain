# TransactionChain
Simple TransactionChain (Transaction Chain), based on Blockchain technology.

Still on development.

Your contributions are welcome.


## System requirements
* PHP 7.0+


## Installation
From command line:
```cmd
php main.php
```
Or run:
```bat
run.bat
```
